# Turon Project

## Описание
Проект для обработки заявок и управления статусами.

## Структура проекта
- public/ - HTML файлы
- src/
  - js/ - JavaScript файлы
  - css/ - CSS стили
  - assets/
    - images/ - Изображения

## Как использовать
1. Откройте public/index.html в браузере
2. ...

## Функционал
- Подключения
- Перевод звонков
- Расторжение договора
- Свитч офф
- Компенсация
- Статус заявки
- Одностороннее прощение
